# sample3
Name
====
ガウシアンフィルタによる処理

Overview

このツールでは、カメラ画像を取得して、ガウシアンフィルタをかけることができる。また、トラックバーをつけることにより、フィルタのかけ具合をインタラクティブに変更できる。

## Description

具体的には、cv2.GaussianBlurコマンドを用いて、引数をトラックバーと連動させることにより実装した。
